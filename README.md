# Chrome Extension Newtab

Chrome Extension을 이용하여 Chrome 브라우저의 새탭 페이지를 변경할 수 있습니다.

이 기능을 이용하여 새탭에 원하는 기능들을 추가할 수 있습니다.

## > Example

아래 사진처럼 Chrome 브라우저의 새탭을 열게 되면, 필요한 정보를 크롤링 하여 출력하게 됩니다.
